🎨 Design Principles

  • Color Palette: Soft blues (#0366D6), neutral grays (#24292E, #586069, #E1E4E8), clean whites
  • Typography: System fonts (SF Pro, Segoe UI), clear hierarchy, 13-16px base sizes
  • Spacing: Generous padding (24px), 20px gaps, breathing room throughout
  • Components: Minimal borders (1px), subtle shadows, 6-8px border radius
  • Interaction: Smooth focus states, border color changes on interaction
