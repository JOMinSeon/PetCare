# AI 반려동물 사료 분석 스킬

## 개요

반려동물 사료의 성분·영양을 수의학 권장치 기준으로 분석하고, 전문적인 Word(.docx) 리포트를 생성한다.
모든 반려동물(개, 고양이, 소동물 등)을 지원하며, 다양한 입력 형태(이미지, 텍스트, 수치, 제품명)를 처리한다.

---

## 입력 처리

### 지원하는 입력 유형

| 입력 유형 | 처리 방법 |
|-----------|-----------|
| 사료 이미지/사진 | 이미지에서 성분표 및 영양정보 OCR/시각 분석 |
| 성분표 텍스트 | 직접 파싱하여 성분 추출 |
| 영양성분 수치 | 수치를 그대로 사용 |
| 제품명 | 웹 검색으로 성분 정보 조회 (web_search 도구 사용) |

### 반려동물 정보 수집 (없으면 사용자에게 질문)
- **종**: 개 / 고양이 / 소동물(토끼·햄스터 등) / 기타
- **나이**: 퍼피·키튼(0~1세) / 성체(1~7세) / 시니어(7세+)
- **체중**: kg 단위 (없으면 평균값으로 분석)
- **특이사항**: 알레르기, 질환 여부 (선택)

---

## 분석 기준 (수의학 권장치)

영양 기준은 `/references/nutrition-standards.md`를 참고한다.

### 핵심 평가 항목

1. **영양성분 평가** — 단백질/지방/탄수화물/수분 함량이 권장 범위 내인지 확인
2. **유해 성분 경고** — 반려동물에게 독성이 있거나 과잉 시 위험한 성분 탐지
3. **나이/체중별 적합성** — 생애 단계에 따른 영양 요구량 대비 평가
4. **타 제품 비교** — 동일 카테고리 제품 대비 상대적 품질 평가

---

## 리포트 생성 (Word .docx)

docx 스킬을 참고하여 Word 문서를 생성한다. `npm install -g docx`가 필요하다.

### 리포트 구조

```
1. 표지
   - 제품명, 분석 날짜, 반려동물 정보
   - 종합 평점 (⭐ 1~5)

2. 요약 섹션
   - 한 눈에 보는 분석 결과 (표)
   - 주요 강점 / 주의사항

3. 영양성분 상세 분석
   - 단백질 / 지방 / 탄수화물 / 수분 평가 표
   - 권장치 대비 현황 (부족/적정/과잉)

4. 유해 성분 경고
   - ⚠️ 위험 등급별 성분 목록 (높음/중간/낮음)
   - 각 성분의 위험 이유 설명

5. 나이/체중별 적합성 평가
   - 현재 반려동물에게 적합한지 종합 판단
   - 급여량 가이드라인 (선택)

6. 타 제품 비교
   - 동일 카테고리 제품 2~3개와 비교 표
   - 가격 대비 품질 코멘트

7. 종합 의견 및 권고사항
   - 전문가 의견 스타일의 종합 평가
   - 개선 권고 또는 대안 제품 제안
```

### 리포트 색상 가이드라인
- 표지/헤더: 진한 초록 `2E7D32` (반려동물/건강 이미지)
- 경고 섹션: 주황 `E65100` (유해 성분)
- 긍정 항목: 연초록 `81C784`
- 주의 항목: 노랑 `FFF9C4`

---

## 실행 워크플로우

```
1. 입력 확인 → 부족한 정보 있으면 사용자에게 질문
2. 성분 추출/파싱
   - 이미지: 시각 분석으로 텍스트 추출
   - 제품명: web_search로 성분 정보 검색
3. 영양 분석 수행 (references/nutrition-standards.md 기준)
4. 유해 성분 스캔 (references/harmful-ingredients.md 기준)
5. 타 제품 비교 데이터 수집 (web_search 활용)
6. docx 스크립트로 Word 리포트 생성
7. 파일을 /mnt/user-data/outputs/에 저장 후 present_files로 제공
```

---

## 참고 파일

- `references/nutrition-standards.md` — 종/나이별 수의학 영양 권장치
- `references/harmful-ingredients.md` — 반려동물 유해 성분 목록 및 위험도

---

## 주의사항

- 본 분석은 **참고용**이며, 건강 문제가 있는 반려동물은 수의사 상담을 권장한다.
- 성분 정보가 불완전할 경우 분석 한계를 리포트에 명시한다.
- 특정 제품을 비방하는 표현 대신 객관적·수치적 평가를 사용한다.
